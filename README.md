## Google Books

Book blogger. Post book, title, synoposis. Save posts & delete posts. 


```
Run  on command line: npm start

```

### Installing
Clone this repo to your local machine using https://github.com/BFronz/BookBlogger.git
 
```
npm install package.json

```
